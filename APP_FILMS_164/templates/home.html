Explication du projet

Pour commencer, le projet consistera à noter toutes les plantes qui ont poussé dans le jardin de ma grand-maman.
Pour l'instant, nous avons inscrit les plantes dont nous nous sommes souvenus avoir planté, et nous pensons ajouter davantage de données dans les années à venir.
Ma grand-maman a toujours voulu avoir un endroit où elle pourrait retrouver les plantes qu'elle avait déjà plantées, donc ce projet a été idéal pour réaliser ce souhait qu'elle avait depuis longtemps.




Comment faire fonctionner le projet ? (même si vous avez déja reussi si vous êtes là)

Donc, avant de commencer, il faut installer un environnement de développement SQL comme Laragon, par exemple, ainsi qu'un IDE pour Python, comme PyCharm avec Python.
Ensuite, on peut exécuter le fichier 1_ImportationDumpSql.py qui est situé dans le dossier database.
À présent, on peut exécuter le fichier run_mon_app.py et cliquer sur le lien affiché dans le terminal.
Et voilà, maintenant que vous êtes sur le site, sentez-vous libre de faire des expériences sur les différentes tables des plantes.
